Tutorial: Unit Upgrades and Promotions

TUTORIAL_UNIT_UPGRADES

Help

Units can receive both Upgrades and Promotions. Upgrades convert a Unit to a different type. For example, a Warrior can be Upgraded to an Axeman. Promotions provide various combat buffs. Both Promotions and Upgrades cost Training. Upgrades can only be performed inside of friendly territory while Promotions can be done anywhere.
