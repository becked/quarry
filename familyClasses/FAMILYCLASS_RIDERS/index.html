Riders

FAMILYCLASS_RIDERS

Advice Found

Riders are a military Family with a focus on Mounted Units. For easy access to Mounted Units, Rider Cities have access to the Import Horses, Import Elephants, and Import Camels Projects, each of which can be completed once per player. Rider Cities are always Connected, which makes Riders a good choice for locations that would be harder to connect by Roads.
