Agent Network

CONCEPT_AGENT_NETWORK

Help

Once you have a Spymaster, a Scout can create an Agent Network in a foreign city. The Scout will remain unavailable for at least turns after, more if damaged.

A character can then be sent to the city to serve as an Agent, gaining yields, visibility of the city's territory, and access to missions like Treachery.
