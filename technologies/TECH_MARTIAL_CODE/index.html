Martial Code

TECH_MARTIAL_CODE

Advice

The Martial Code technology unlocks the building of Towers, further enhancing the defenses of a City with an existing Moat. It also opens the option of militarizing your Religion through the Holy War Law to improve new recruits, or gaining the more peaceful economic and scientific benefits of the Pilgrimage Law.
